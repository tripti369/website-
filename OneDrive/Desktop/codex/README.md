# HackMates

HackMates is a static hackathon collaboration website where students can:

- create a profile with name, college roll number, course, year, GitHub, and interest field
- browse other student profiles
- open another student's GitHub profile
- approach teammates by sending a collaboration request inside the website

## Run

Open `index.html` in any browser.

## Notes

- Profiles and collaboration requests are stored in the browser using `localStorage`.
- The app ships with sample student profiles so the discovery flow works immediately.
